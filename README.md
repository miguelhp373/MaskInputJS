## MascÃ¡ras
---
- Telefone :

    `(11)91111-1111`
    <br/>
    ```html
    <input type='text' class = 'inputnumberphoneformat'/>
    ``` 

- CPF :
&nbsp;
    `111.111.111-11`
    <br/>
    ```html

    ```

&nbsp;
    `00/00/0000`

    <br/>
    ```html
    <input type='text' class = 'inputdateformat_ddmmaaaahhmm'/>
    ```
